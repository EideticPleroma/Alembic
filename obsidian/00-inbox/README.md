# Inbox

Drop quick notes, ideas, and captures here. Process regularly into appropriate locations.

## Quick Capture Template

```markdown
## [Topic]
**Date**: {{date}}
**Type**: idea | question | resource | bug | feature

[Content here]

**Action**: 
**Move to**: 
```

## Processing Guide

When processing inbox items:

1. **Ideas** -> Move to `02-project/ideas.md` or create feature request
2. **Questions** -> Research and document answer, or ask in appropriate channel
3. **Resources** -> Move to `03-knowledge/` with summary
4. **Bugs** -> Create task in `02-project/tasks.md`
5. **Features** -> Add to backlog in `02-project/kanban.md`
