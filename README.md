# Alembic

> The vessel of transformation - where questions become insight.

Alembic is a Hermetic AI-powered tarot reading application that serves as a digital mirror for self-reflection. Using the symbolism of tarot through the lens of depth psychology and Hermetic philosophy, Alembic helps users explore their inner landscape without fortune-telling or prediction.

## Features

- **AI-Powered Readings**: Interpretations generated by Grok, grounded in Hermetic principles
- **Multiple Spreads**: Three-card, Shadow Work, Celtic Cross (more coming)
- **Follow-up Conversations**: Deepen your understanding through dialogue
- **Reading History**: Track your journey over time
- **Privacy-First**: Your questions and readings are yours alone

## Quick Start

### Prerequisites

- Node.js 20+
- Python 3.11+
- Docker (optional, for Ollama)

### Development Setup

```bash
# Clone the repository
git clone https://github.com/yourusername/alembic.git
cd alembic

# Backend setup
cd backend
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -e ".[dev]"
cp .env.example .env
# Edit .env with your credentials

# Frontend setup
cd ../frontend
npm install
cp .env.example .env.local
# Edit .env.local with your credentials

# Start development servers
# Terminal 1:
cd backend && uvicorn app.main:app --reload

# Terminal 2:
cd frontend && npm run dev
```

### Local LLM (Free Development)

```bash
# Install Ollama
curl -fsSL https://ollama.com/install.sh | sh

# Pull a model
ollama pull llama3.2

# Ollama runs on localhost:11434
```

## Philosophy

Alembic is built on three pillars:

1. **Hermetic Foundation**: The Seven Hermetic Principles guide interpretation
2. **Depth Psychology**: Jungian archetypes inform card meanings
3. **Ethical Practice**: No predictions, no fear-mongering, no false certainty

The cards are mirrors, not crystal balls.
